# Event-Registration-Form
A simple and responsive Event Registration Form built using HTML, CSS, and JavaScript.

## 🚀 Features
- User-friendly form design
- Input fields for name, email, phone, and more
- Dropdown selection (Gender, Event, Mode)
- Confirmation checkbox validation
- Displays registered participants in a table dynamically

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (inside HTML)

## 📂 Project Structure
- index.html → Structure + JavaScript logic
- style.css → Styling


## 🌐 Live Demo
https://balajidev-cloud.github.io/Event-Registration-Form/


## 📌 How to Use
1. Open the project
2. Fill the form
3. Click Submit
4. Data will be displayed in the table
